Monday

Goal(s):

  • Khan Academy's "More HTML tags"
What I learned?
  1. Adding links to a page with the anchor tag < a > < /a > and hyper reference attribute href="insert your link here"
  2. Using the hyper reference attribute to move up and down the same webpage (by referencing ids)
  3. Creating tables using the following tags:
    A. < table > < /table > (required to access tables)
    B. < thead > < /thead > (header column)
    C. < tr > < /tr > (rows)
    D. < th > < /th > (header cells)
  4. Commenting out code with <--! "insert your comment here" -->
